# Исследование архитектуры KAN и ее модификаций для задач распознавания образов

Репозиторий содержит материалы выпускной квалификационной работы, посвящённой исследованию архитектуры **Kolmogorov–Arnold Networks (KAN)** и её модификаций в задачах классификации изображений. В работе рассматриваются как базовая KAN, так и гибридные архитектуры, а также проводится сравнительный анализ их эффективности на нескольких наборах данных.

## О проекте

Цель работы — исследовать, насколько архитектура KAN и её модификации подходят для задач распознавания образов, и сравнить их с традиционными нейросетевыми подходами.  
В рамках проекта реализованы:
- теоретическое исследование архитектуры KAN и её математических основ;
- разработка и обучение нескольких вариантов моделей;
- экспериментальное сравнение моделей на разных датасетах;
- анализ качества по метрикам classification performance;
- визуализация архитектур и результатов обучения;
- сохранение моделей, чекпоинтов и промежуточных артефактов.


## Модели и подходы

В работе рассматриваются следующие архитектурные подходы:
- базовая KAN;
- модификации KAN;
- гибридные архитектуры на основе KAN;
- сравнение KAN-подхода с классическим MLP;

## Экспериментальные данные

Эксперименты проводились не на одном, а на нескольких наборах данных, что позволяет оценить устойчивость архитектур в задачах разной сложности.  
Среди используемых датасетов:
- STL-10;
- Caltech-101;
- Food-101;
- ChestX-ray14.

Такой набор данных даёт возможность сравнивать модели как на относительно простых, так и на более сложных задачах распознавания образов.

## Результаты

По результатам экспериментов оцениваются метрики качества классификации, включая:
- Accuracy;
- Precision;
- Recall;
- F1-score.

Для отдельных экспериментов также анализировались вычислительные характеристики и поведение функции потерь в процессе обучения.

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/aaastrakhantseva/VKR_KAN.git
cd VKR_KAN
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate
```

Для Windows:

```bash
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

Для стабильной работы `pykan` рекомендуется использовать `sympy==1.11.1`.

## Запуск

Основной экспериментальный ноутбук:

```bash
jupyter notebook notebooks/VKR_KAN.ipynb
```

Также проект можно запускать в **Google Colab**.
